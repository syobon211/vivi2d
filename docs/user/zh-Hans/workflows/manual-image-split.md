# 手动图像分割

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

当单张图需要在 Auto Setup 前拆成部件时，请使用手动蒙版。

## 你需要准备

- 一份单张图或部分合并素材的副本。
- 对头发、脸、眼睛、嘴、身体和配饰等重要区域的判断。
- 在应用分割前查看警告的时间。

## 步骤

1. 从包含源素材的项目中打开手动图像分割工作区。
2. 先为主要部件创建蒙版，再细化较小的细节。
3. 根据要选择的边缘使用画笔或套索工具。平滑的套索路径适合头发和布料轮廓。
4. 保持脸、眼睛和嘴区域干净，避免它们被误加入会移动的头发或配饰蒙版。
5. 检查空蒙版、重叠、不透明像素丢失和小碎片等质量警告。
6. 只有当蒙版确实代表后续工具要使用的部件时，才应用分割。

## 检查结果

已接受的分割应生成有用的图层区域，不应隐藏重要像素，也不应留下明显的重复轮廓。

## 如果出现问题

如果警告提示需要手动检查，请先修正蒙版再应用。预览用的底层补全不会自动保存，只有已接受的清理结果才会进入项目。
