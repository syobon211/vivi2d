# 导出

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

请在编辑器和 Viewer 中检查项目后再导出。

## 你需要准备

- 没有阻塞性审查警告的已保存项目。
- 不会覆盖源素材的目标文件夹。
- 明确的目标流程，例如预览、分享或集成测试。

## 步骤

1. 打开导出控制前先保存项目。
2. 选择与测试流程匹配的导出目标。
3. 阅读应用显示的实验性或兼容性说明。
4. 导出到一个新文件夹，方便检查结果。
5. 分享前，请在目标 Viewer 或示例中打开导出结果。

## 检查结果

导出的文件应完整、易于识别，并且不需要私人工作目录中的文件就能在目标流程中加载。

## 如果出现问题

如果导出结果缺少纹理、图层或动作，请返回项目检查源数据后再导出。保留一个已知正常的导出结果，便于比较。
