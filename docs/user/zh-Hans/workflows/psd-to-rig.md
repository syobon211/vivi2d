# 从素材到起始绑定

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

不要跳过审查步骤，把源素材推进到可编辑的起始设置。

## 你需要准备

- 能够分别检查角色部位的素材。
- 已保存的项目副本。
- 在应用生成结果前查看警告的时间。

## 步骤

1. 打开素材，确认画布、图层和属性面板正确。
2. 如果素材已经分层，请检查图层名称、可见性和顺序。
3. 如果素材是单张图，请先使用[手动图像分割](manual-image-split.md)创建已接受的蒙版。
4. 当图层或蒙版与预期角色部位一致后，再运行 [Auto Setup](auto-setup.md)。
5. 应用前先检查 Auto Setup 会保存哪些内容。
6. 在 [Viewer](viewer.md) 中预览结果；如果动作或图层分离不自然，请回到前一步。

## 检查结果

项目应包含可编辑图层、已审查的起始设置，并且没有被应用步骤隐藏的阻塞警告。

## 如果出现问题

不要在混乱输入上反复应用 Auto Setup。请回到图层名称、蒙版或手动清理步骤，然后从干净的项目副本重新运行。
