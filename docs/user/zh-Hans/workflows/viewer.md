# 在 Viewer 中预览

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

使用 Viewer 检查模型在编辑工作区之外的表现。

## 你需要准备

- Viewer 可以加载的已保存 Vivi2D 项目或已导出的模型文件。
- 一个便于检查轮廓的简单背景。
- 测试表情、参数和基础动作的时间。

## 步骤

1. 从应用或相关工作流中打开 Viewer。
2. 加载要检查的模型。
3. 先使用中性背景，这样更容易发现重复轮廓和隐藏区域露出。
4. 在测试动作或表情前，先检查静止状态。
5. 尝试可用控制，然后重置到静止状态，确认模型能干净地返回。
6. 如果 Viewer 显示警告，请在导出前把它们当作审查任务处理。

## 检查结果

模型应能加载、响应控制，并在没有明显缺失部件或异常轮廓的情况下回到稳定静止状态。

## 如果出现问题

如果 Viewer 显示空白，请回到编辑器确认模型是否正确保存或导出。如果像是显示问题，请阅读[显示与 GPU](../troubleshooting/display-and-gpu.md)。
