# Auto Setup

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

在确认哪些内容会保存、哪些只是预览的同时，创建一个起始设置。

## 你需要准备

- 已分离的图层，或已接受的手动分割蒙版。
- 准备步骤中没有阻塞性的质量警告。
- 一个可回退的已保存项目副本。

## 步骤

1. 图层结构准备好后打开 Auto Setup。
2. 根据你需要的控制程度，选择初学者或高级检测级别。
3. 应用前先检查生成的计划。
4. 阅读保存操作区域。这些操作可能包括骨骼、静态权重、物理组、清理摘要和数值设置，并会成为项目数据。
5. 阅读丢弃预览区域。临时预览信息和诊断细节不会保存到项目中。
6. 检查关于受保护区域、重复轮廓、隐藏区域露出和未完成检查的警告。
7. 只有当保存操作和警告符合预期时才应用。

## 检查结果

项目应包含清晰、可继续编辑的起始设置。审查面板应说明哪些内容已保存、哪些已丢弃、哪些仍需要手动处理。

## 如果出现问题

如果建议移动了错误区域、脸部变化过大，或要求手动检查，请取消而不是应用。修正源图层或蒙版后重新运行 Auto Setup。
