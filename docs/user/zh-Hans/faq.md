# FAQ

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

在阅读完整工作流页面前，可以先用这些回答快速了解方向。

## Vivi2D 已适合公开生产使用了吗？

项目仍在准备公开发布。在发布说明明确之前，请把当前构建和文档视为草稿。

## 我应该使用哪种语言？

公开默认 UI 是英文。用户界面和文档也提供日语、简体中文和韩语，但最终母语审校可能会结合公开后的反馈继续推进。

## 应该先用手动图像分割还是 Auto Setup？

如果素材是单张图，请先使用手动图像分割。如果素材已经有清晰图层，请先审查图层，再尝试 Auto Setup。

## Auto Setup 会保存什么？

Auto Setup 会保存已审查的项目操作，例如骨骼、静态权重、物理组、清理摘要和数值设置。临时预览计算和诊断细节会被丢弃。

## 外部工具可以控制 Viewer 吗？

可以，通过本地 Viewer API 流程进行。请完成配对、检查作用域，并在测试结束后关闭或撤销会话。
