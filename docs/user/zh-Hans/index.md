# Vivi2D 用户指南

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

本指南面向画师、绑定师、主播和 SDK 用户，帮助你在不了解仓库内部结构的情况下理解 Vivi2D 当前的使用流程。

## 从这里开始

1. 阅读[安装 Vivi2D](getting-started/install.md)，了解当前发布状态。
2. 继续阅读[首次启动](getting-started/first-launch.md)，确认默认语言、深色主题和空工作区。
3. 通过[打开第一个项目](getting-started/open-your-first-project.md)导入素材。
4. 按照[从素材到初始绑定](workflows/psd-to-rig.md)了解完整制作路径。

## 主要流程

- [手动图像拆分](workflows/manual-image-split.md) 会把扁平图像整理成更适合设置的部件。
- [Auto Setup](workflows/auto-setup.md) 会从准备好的图层生成可审核的初始设置。
- [在 Viewer 中预览](workflows/viewer.md) 用于在导出或集成前检查模型。
- [Viewer API 基础](workflows/viewer-api.md) 说明外部工具的本地配对流程。

## 可选集成

- [ComfyUI 设置](integrations/comfyui.md) 说明如何连接本机 ComfyUI，并保持 Vivi2D 的审核步骤。

## 文档状态

这些页面仍是公开发布前的用户文档草稿。最终截图和视频会在之后的媒体整理阶段加入，因此每个页面都尽量做到只看文字也能理解。
