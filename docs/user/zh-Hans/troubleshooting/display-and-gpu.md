# 显示与 GPU

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

当应用已打开，但画布或预览显示不正确时，请使用这些步骤。

## 你需要准备

- 一个没有导入错误的项目。
- 如果当前项目很大，请准备一个小测试文件。
- 可以查看系统显示设置。

## 步骤

1. 确认图层列表中有可见图层。
2. 在判断画布为空之前，先尝试适配视图或重置视图。
3. 使用较小的已知正常项目，把渲染问题和项目数据问题分开。
4. 关闭其他占用 GPU 较多的应用，然后重新打开 Vivi2D。
5. 如果问题只出现在外接显示器上，请在主显示器上测试。

## 检查结果

缩放、平移或更改选择后，画布应重新绘制。Viewer 应在简单背景上显示模型。

## 如果出现问题

报告显示问题时，请描述可见症状和大致硬件类别。除非已打码，否则不要包含私人素材截图。
