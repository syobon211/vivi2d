# 语言与文字显示

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

当标签、菜单或对话框与所选语言不一致时，请使用本页。

## 你需要准备

- 当前应用语言设置。
- 文字显示异常的屏幕或对话框。
- 简短说明问题是未翻译、文字被截断，还是字符不可读。

## 步骤

1. 打开设置，确认当前选择的语言。
2. 切换到另一种语言，再切回你想使用的语言。
3. 重新打开显示异常的对话框或面板。
4. 如果问题仍然存在，记录可见标签以及页面或面板名称。
5. 如果需要报告布局截图，请使用合成截图或裁剪，并隐藏私人素材。

## 检查结果

所选语言应应用到菜单、对话框、工作流面板和常见操作标签。

## 如果出现问题

开发过程中，某些草稿页面或新 UI 可能仍有英文回退。报告时请提供可见标签和语言，不要包含本地路径或私人项目数据。
