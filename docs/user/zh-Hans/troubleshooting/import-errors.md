# 导入错误

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

判断问题来自文件类型、文件大小、访问权限，还是意外的源数据。

## 你需要准备

- 无法打开的文件。
- 一个较小且已知正常的测试文件。
- 一个安全的项目文件夹，不要使用临时下载位置。

## 步骤

1. 先尝试打开较小的已知正常图像。
2. 检查扩展名是否与实际文件内容一致。
3. 把测试副本移动到普通项目文件夹后重试。
4. 如果源文件很大，请尝试较小副本以确认流程。
5. 如果只有一个源文件失败，请保留原文件不变，并使用副本继续排查。

## 检查结果

已知正常文件应能打开。如果只有一个源文件失败，问题可能在该源文件或当前位置。

## 如果出现问题

不要在公开报告中粘贴本地堆栈、私人文件名或完整文件路径。请描述文件类型、大致大小和失败的可见操作。
