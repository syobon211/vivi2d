# 打开第一个项目

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

先打开一份安全的测试副本，在开始修改前检查编辑器状态。

## 你需要准备

- 要测试的素材副本。
- Vivi2D 当前支持的文件格式。
- 在运行设置工具前检查图层列表和画布的时间。

## 步骤

1. 从文件菜单打开或导入测试素材。
2. 等待画布更新完成，再点击其他工具。
3. 检查图层列表。带有部位名称的图层会比通用名称更容易审查。
4. 选择一个图层，确认属性面板会更新。
5. 缩放和平移画布，确认素材可见并且位置正确。
6. 在运行手动图像分割或 Auto Setup 之前，先保存项目副本。

## 检查结果

画布应显示素材，图层列表应包含导入的部件，选择图层时属性面板应有响应。

## 如果出现问题

如果文件无法打开，请先尝试一个较小的已知正常文件，并查看[导入错误](../troubleshooting/import-errors.md)。如果图层存在但画布为空，请查看[显示与 GPU](../troubleshooting/display-and-gpu.md)。
