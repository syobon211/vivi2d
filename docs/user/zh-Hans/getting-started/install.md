# 安装 Vivi2D

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

请使用当前最可靠的方式启动你手中的 Vivi2D。

## 你需要准备

- 一台可以运行现代桌面应用的电脑。
- 可信的 Vivi2D 构建来源，或本地开发仓库。
- 一个用于测试项目的文件夹，不要和原始素材混在一起。

## 步骤

1. 如果官方打包版本已经可用，请从项目发布来源下载，并保留原始压缩包名称。
2. 如果暂时没有打包版本，请按照仓库 README 中的本地开发说明操作，不要从非官方镜像下载。
3. 从普通用户文件夹启动 Vivi2D，不要直接在临时下载目录中运行。
4. 在导入测试项目前，先复制一份原始素材。
5. 正式工作前先启动一次，确认窗口标题、主题和语言设置。

## 检查结果

Vivi2D 应该打开到一个空的编辑器窗口。公开默认界面为英文，并在未更改设置时使用深色主题。

## 如果出现问题

如果应用无法打开，请换到没有特殊权限限制的文件夹，并检查系统是否阻止了下载的应用。报告问题时不要公开私人素材或本地文件路径。
