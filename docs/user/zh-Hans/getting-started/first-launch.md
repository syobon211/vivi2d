# 首次启动

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

在打开项目之前，先确认空工作区代表什么。

## 你需要准备

- 已安装的 Vivi2D，或从开发构建启动的 Vivi2D。
- 几分钟时间检查默认工作区。

## 步骤

1. 启动 Vivi2D，等待编辑器窗口加载完成。
2. 确认应用标题显示为 Vivi2D Editor。
3. 注意公开默认 UI 为英文，默认主题为深色。
4. 如果想切换到日语、简体中文或韩语，请打开设置菜单。
5. 查看主要区域：画布、图层列表、属性面板、参数和底部时间线区域。
6. 如果画布为空，不用担心。这是未加载项目时的正常状态。

## 检查结果

你应该看到一个稳定的编辑器窗口，且尚未加载项目。只有在尝试打开素材后，才应该出现文件相关错误。

## 如果出现问题

如果文字被截断或语言混杂，请切换一次语言，必要时重启应用，并查看[语言与文字显示](../troubleshooting/localization.md)。
