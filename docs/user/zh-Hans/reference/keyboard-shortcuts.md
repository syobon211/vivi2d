# 键盘快捷键

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

快捷键用于加快操作，不应成为执行某个动作的唯一方式。

## 常见区域

- 撤销和重做用于从编辑错误中恢复。
- 工具切换会改变当前编辑模式。
- 图层移动会改变堆叠顺序，请谨慎使用。
- 保存和另存为快捷键不能替代定期项目备份。

## 自定义

从设置菜单打开快捷键设置。如果快捷键与操作系统、浏览器、辅助技术或输入设备冲突，请选择其他绑定，或使用界面上的按钮。

## 检查结果

快捷键设置对话框应显示本地化的操作名称和稳定的按键绑定。更改快捷键后，请在项目副本中测试。

## 如果出现问题

如果快捷键没有反应，请先点击可见控件。然后检查是否有其他应用或操作系统占用了该按键。
