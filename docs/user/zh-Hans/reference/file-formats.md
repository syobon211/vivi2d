# 文件格式

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

请根据正在测试的流程选择输入和输出文件。

## 支持的输入

公开前支持情况可能变化，因此请以当前应用 UI 为最终依据。总体来说，Vivi2D 流程关注可以变成可编辑图层或已接受蒙版的素材。

## 合适的源素材

- 导入前保留原始素材副本。
- 分层素材应优先使用清晰的图层名称。
- 单张图通常需要在 Auto Setup 前使用手动图像分割。
- 不要在公开错误报告或截图中使用私人委托素材。

## 输出文件

分享前，请在目标 Viewer 或示例中测试导出的文件。如果某个输出路径标为实验性，请在发布说明明确前把它当作测试产物。

## 如果出现问题

如果文件扩展名和实际内容不一致，导入可能失败。安全恢复步骤请查看[导入错误](../troubleshooting/import-errors.md)。
