# 设置

## 翻译草稿说明

本页是翻译草稿。最新且最详细的步骤会先反映在同一主题的英文版中。公开发布前，本翻译也会扩充并重新审核。

设置用于让工作区更易阅读，而不改变项目数据。

## 常见设置

- 语言会改变拥有该设置的编辑器或 Viewer 界面的 UI 语言。
- 主题会改变工作区外观。
- 工作流偏好可能影响面板和审查辅助的显示方式。
- 快捷键设置会独立于可见按钮管理键盘绑定。

## 语言说明

公开默认 UI 是英文。当前本地化集合包含日语、简体中文和韩语。如果切换语言后仍有英文标签，请把它视为本地化问题，而不是用户操作错误。

## 检查结果

更改设置后，打开菜单或对话框确认新选择已显示。某些设置可能需要重新打开相关窗口。

## 如果出现问题

如果文字乱码、被截断或语言混杂，请查看[语言与文字显示](../troubleshooting/localization.md)。
