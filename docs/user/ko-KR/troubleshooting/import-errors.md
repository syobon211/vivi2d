# 가져오기 오류

## 번역 초안 안내

이 페이지는 번역 초안입니다. 가장 최신이고 자세한 절차는 같은 주제의 영어판에 먼저 반영됩니다. 공개 전에는 이 번역도 내용을 확장하고 다시 검토합니다.

문제가 파일 형식, 파일 크기, 접근 권한, 예상하지 못한 원본 데이터 중 어디에 있는지 확인합니다.

## 필요한 것

- 열리지 않은 파일.
- 작고 정상적으로 열리는 테스트 파일.
- 임시 다운로드 위치가 아닌 안전한 프로젝트 폴더.

## 단계

1. 먼저 작고 정상인 이미지를 열어 봅니다.
2. 파일 확장자가 실제 파일 내용과 맞는지 확인합니다.
3. 테스트 사본을 일반 프로젝트 폴더로 옮기고 다시 시도합니다.
4. 원본이 크다면 작은 사본으로 워크플로를 확인합니다.
5. 특정 원본만 실패하면 원본은 그대로 두고 사본으로 작업합니다.

## 결과 확인

정상 파일은 열려야 합니다. 특정 원본만 실패한다면 문제는 그 원본이나 현재 위치에 있을 가능성이 큽니다.

## 문제가 있을 때

공개 보고서에 로컬 스택 추적, 비공개 파일 이름, 전체 파일 경로를 붙여 넣지 마세요. 파일 형식, 대략적인 크기, 실패한 화면 동작을 설명하세요.
