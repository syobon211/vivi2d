# 언어와 문자 표시

## 번역 초안 안내

이 페이지는 번역 초안입니다. 가장 최신이고 자세한 절차는 같은 주제의 영어판에 먼저 반영됩니다. 공개 전에는 이 번역도 내용을 확장하고 다시 검토합니다.

레이블, 메뉴, 대화상자가 선택한 언어와 맞지 않을 때 사용합니다.

## 필요한 것

- 현재 앱 언어 설정.
- 텍스트가 이상하게 보이는 화면이나 대화상자.
- 문제가 번역 누락, 텍스트 잘림, 읽을 수 없는 문자 중 무엇에 가까운지에 대한 짧은 설명.

## 단계

1. 설정을 열어 선택된 언어를 확인합니다.
2. 다른 언어로 바꾼 뒤 원하는 언어로 다시 바꿉니다.
3. 문제가 보였던 대화상자나 패널을 다시 엽니다.
4. 문제가 남아 있으면 보이는 레이블과 페이지 또는 패널 이름을 기록합니다.
5. 레이아웃 보고에 스크린샷이 필요하면 비공개 아트워크를 숨긴 합성 이미지나 잘라낸 이미지를 사용합니다.

## 결과 확인

선택한 언어가 메뉴, 대화상자, 워크플로 패널, 공통 동작 레이블에 적용되어야 합니다.

## 문제가 있을 때

개발 중에는 일부 초안 페이지나 새 UI에 영어가 남을 수 있습니다. 보고할 때는 보이는 레이블과 언어를 알려 주고, 로컬 경로나 비공개 프로젝트 데이터는 포함하지 마세요.
