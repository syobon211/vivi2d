# 표시와 GPU

## 번역 초안 안내

이 페이지는 번역 초안입니다. 가장 최신이고 자세한 절차는 같은 주제의 영어판에 먼저 반영됩니다. 공개 전에는 이 번역도 내용을 확장하고 다시 검토합니다.

앱은 열리지만 캔버스나 미리보기가 올바르게 보이지 않을 때 사용합니다.

## 필요한 것

- 가져오기 오류 없이 열린 프로젝트.
- 현재 프로젝트가 크다면 작은 테스트 파일.
- 시스템 디스플레이 설정을 확인할 수 있는 상태.

## 단계

1. 레이어 목록에 보이는 레이어가 있는지 확인합니다.
2. 캔버스가 비었다고 판단하기 전에 맞춤 보기나 보기 재설정을 시도합니다.
3. 작고 정상인 프로젝트를 사용해 렌더링 문제와 프로젝트 데이터 문제를 분리합니다.
4. GPU를 많이 사용하는 다른 앱을 닫고 Vivi2D를 다시 엽니다.
5. 외부 모니터에서만 문제가 보이면 기본 디스플레이에서도 테스트합니다.

## 결과 확인

확대, 이동, 선택 변경 후 캔버스가 다시 그려져야 합니다. Viewer는 단순한 배경 위에 모델을 표시해야 합니다.

## 문제가 있을 때

표시 문제를 보고할 때는 보이는 증상과 대략적인 하드웨어 종류를 설명합니다. 비공개 아트워크 스크린샷은 가린 뒤에만 사용하세요.
