# FAQ

## 번역 초안 안내

이 페이지는 번역 초안입니다. 가장 최신이고 자세한 절차는 같은 주제의 영어판에 먼저 반영됩니다. 공개 전에는 이 번역도 내용을 확장하고 다시 검토합니다.

전체 워크플로 문서를 읽기 전에 빠르게 방향을 잡고 싶을 때 사용합니다.

## Vivi2D는 공개 제작 작업에 바로 사용할 수 있나요?

프로젝트는 아직 공개 릴리스를 준비 중입니다. 릴리스 노트에 명시되기 전까지 현재 빌드와 문서는 초안으로 다루세요.

## 어떤 언어를 사용해야 하나요?

공개 기본 UI는 영어입니다. 사용자 UI와 문서는 일본어, 중국어 간체, 한국어도 제공하지만, 최종 원어민 검토는 공개 후 피드백을 포함해 진행될 수 있습니다.

## Manual Image Split과 Auto Setup 중 무엇부터 시작하나요?

아트워크가 한 장이면 수동 이미지 분할부터 시작합니다. 이미 명확한 레이어가 있다면 레이어를 검토한 뒤 Auto Setup을 시도합니다.

## Auto Setup은 무엇을 저장하나요?

Auto Setup은 검토된 프로젝트 작업으로 본, 정적 가중치, 물리 그룹, 정리 요약, 수치 설정 등을 저장합니다. 임시 미리보기 계산과 진단 세부 정보는 버려집니다.

## 외부 도구가 Viewer를 제어할 수 있나요?

로컬 Viewer API 흐름을 통해 가능합니다. 페어링하고 범위를 확인한 뒤, 테스트가 끝나면 세션을 닫거나 취소하세요.
