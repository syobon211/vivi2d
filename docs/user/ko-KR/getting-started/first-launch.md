# 첫 실행

## 번역 초안 안내

이 페이지는 번역 초안입니다. 가장 최신이고 자세한 절차는 같은 주제의 영어판에 먼저 반영됩니다. 공개 전에는 이 번역도 내용을 확장하고 다시 검토합니다.

프로젝트를 열기 전에 빈 작업 공간이 무엇을 뜻하는지 확인합니다.

## 필요한 것

- 설치했거나 개발 빌드에서 실행한 Vivi2D.
- 기본 작업 공간을 살펴볼 몇 분.

## 단계

1. Vivi2D를 시작하고 편집기 창 로딩이 끝날 때까지 기다립니다.
2. 앱 제목이 Vivi2D Editor인지 확인합니다.
3. 공개 기본 UI는 영어이고 기본 테마는 다크입니다.
4. 일본어, 중국어 간체, 한국어로 바꾸려면 설정 메뉴를 엽니다.
5. 캔버스, 레이어 목록, 속성 패널, 파라미터, 아래쪽 타임라인 영역을 살펴봅니다.
6. 캔버스가 비어 있어도 괜찮습니다. 프로젝트가 없는 정상 상태입니다.

## 결과 확인

프로젝트가 열리지 않은 안정적인 편집기 창이 보여야 합니다. 아트워크를 열기 전에는 파일 오류가 표시되지 않아야 합니다.

## 문제가 있을 때

글자가 잘리거나 언어가 섞여 보이면 언어를 한 번 변경하고, 필요하면 앱을 다시 시작한 뒤 [언어와 문자 표시](../troubleshooting/localization.md)를 확인하세요.
