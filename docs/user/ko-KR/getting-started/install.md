# Vivi2D 설치

## 번역 초안 안내

이 페이지는 번역 초안입니다. 가장 최신이고 자세한 절차는 같은 주제의 영어판에 먼저 반영됩니다. 공개 전에는 이 번역도 내용을 확장하고 다시 검토합니다.

현재 가지고 있는 Vivi2D에서 가장 신뢰할 수 있는 실행 경로를 사용합니다.

## 필요한 것

- 최신 데스크톱 앱을 실행할 수 있는 컴퓨터.
- 신뢰할 수 있는 Vivi2D 빌드 또는 로컬 개발 저장소.
- 원본 아트워크와 테스트 프로젝트를 분리해 둘 작업 폴더.

## 단계

1. 공식 패키지 빌드가 있다면 프로젝트의 배포 위치에서 내려받고, 압축 파일 이름은 그대로 보관합니다.
2. 아직 패키지 빌드가 없다면 비공식 미러가 아니라 저장소 README의 개발 빌드 안내를 따릅니다.
3. 임시 다운로드 폴더가 아니라 일반 사용자 폴더에서 Vivi2D를 실행합니다.
4. 테스트 프로젝트에 가져오기 전에 원본 아트워크를 복사해 둡니다.
5. 실제 작업 전에 한 번 실행해 창 제목, 테마, 언어 설정을 확인합니다.

## 결과 확인

Vivi2D가 빈 편집기 창으로 열리면 성공입니다. 공개 기본 UI는 영어이며, 설정을 바꾸지 않으면 다크 테마를 사용합니다.

## 문제가 있을 때

앱이 열리지 않으면 특수 권한이 필요한 폴더를 피하고, 운영체제가 다운로드한 앱을 차단했는지 확인하세요. 문제를 보고할 때는 비공개 아트워크나 로컬 파일 경로를 공유하지 마세요.
