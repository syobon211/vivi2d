# 첫 프로젝트 열기

## 번역 초안 안내

이 페이지는 번역 초안입니다. 가장 최신이고 자세한 절차는 같은 주제의 영어판에 먼저 반영됩니다. 공개 전에는 이 번역도 내용을 확장하고 다시 검토합니다.

안전한 테스트 사본을 열고, 변경을 시작하기 전에 편집기 상태를 확인합니다.

## 필요한 것

- 테스트할 아트워크 사본.
- Vivi2D가 현재 지원하는 형식의 파일.
- 설정 도구를 실행하기 전에 레이어 목록과 캔버스를 확인할 시간.

## 단계

1. 파일 메뉴에서 테스트용 아트워크를 열거나 가져옵니다.
2. 다른 도구를 누르기 전에 캔버스 업데이트가 끝날 때까지 기다립니다.
3. 레이어 목록을 확인합니다. 일반 이름보다 부위 이름이 있는 레이어가 검토하기 쉽습니다.
4. 레이어를 선택하고 속성 패널이 업데이트되는지 확인합니다.
5. 캔버스를 확대, 축소, 이동하여 아트워크가 보이고 위치가 맞는지 확인합니다.
6. 수동 이미지 분할이나 Auto Setup을 실행하기 전에 프로젝트 사본을 저장합니다.

## 결과 확인

캔버스에 아트워크가 보이고, 레이어 목록에 가져온 파트가 있으며, 레이어 선택 시 속성 패널이 반응해야 합니다.

## 문제가 있을 때

파일이 열리지 않으면 더 작은 정상 파일로 먼저 확인하고 [가져오기 오류](../troubleshooting/import-errors.md)를 읽어 보세요. 레이어는 있는데 캔버스가 비어 있으면 [표시와 GPU](../troubleshooting/display-and-gpu.md)를 확인하세요.
