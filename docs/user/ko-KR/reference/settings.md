# 설정

## 번역 초안 안내

이 페이지는 번역 초안입니다. 가장 최신이고 자세한 절차는 같은 주제의 영어판에 먼저 반영됩니다. 공개 전에는 이 번역도 내용을 확장하고 다시 검토합니다.

설정은 프로젝트 데이터를 바꾸지 않고 작업 공간을 읽기 쉽게 만드는 데 사용합니다.

## 공통 설정

- 언어는 해당 설정을 가진 편집기 또는 Viewer 화면의 UI 언어를 바꿉니다.
- 테마는 작업 공간의 시각적 모양을 바꿉니다.
- 워크플로 환경 설정은 패널과 검토 도우미 표시 방식에 영향을 줄 수 있습니다.
- 단축키 설정은 화면 버튼과 별도로 키보드 바인딩을 관리합니다.

## 언어 참고

공개 기본 UI는 영어입니다. 현재 현지화 세트에는 일본어, 중국어 간체, 한국어가 포함됩니다. 언어를 바꾼 뒤에도 영어 레이블이 남아 있다면 사용자 실수가 아니라 현지화 문제로 다루세요.

## 결과 확인

설정을 바꾼 뒤 메뉴나 대화상자를 열어 새 선택이 보이는지 확인합니다. 일부 설정은 관련 창을 다시 열어야 할 수 있습니다.

## 문제가 있을 때

글자가 깨지거나 잘리거나 언어가 섞여 보이면 [언어와 문자 표시](../troubleshooting/localization.md)를 확인하세요.
