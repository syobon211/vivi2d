# 키보드 단축키

## 번역 초안 안내

이 페이지는 번역 초안입니다. 가장 최신이고 자세한 절차는 같은 주제의 영어판에 먼저 반영됩니다. 공개 전에는 이 번역도 내용을 확장하고 다시 검토합니다.

단축키는 작업을 빠르게 하는 보조 수단이며, 어떤 동작에 접근하는 유일한 방법이 되어서는 안 됩니다.

## 공통 영역

- 실행 취소와 다시 실행은 편집 실수에서 돌아올 때 사용합니다.
- 도구 전환은 현재 편집 모드를 바꿉니다.
- 레이어 이동은 쌓임 순서를 바꾸므로 주의해서 사용합니다.
- 저장과 다른 이름 저장 단축키는 정기적인 프로젝트 백업을 대체하지 않습니다.

## 사용자 지정

설정 메뉴에서 단축키 설정을 엽니다. 단축키가 운영체제, 브라우저, 보조 기술, 입력 장치와 충돌하면 다른 키를 선택하거나 화면의 버튼을 사용하세요.

## 결과 확인

단축키 설정 대화상자에는 현지화된 동작 이름과 안정적인 키 바인딩이 보여야 합니다. 단축키를 바꾼 뒤에는 프로젝트 사본에서 테스트합니다.

## 문제가 있을 때

단축키가 반응하지 않으면 먼저 보이는 컨트롤을 클릭합니다. 그다음 다른 앱이나 운영체제가 해당 키를 가져가는지 확인하세요.
