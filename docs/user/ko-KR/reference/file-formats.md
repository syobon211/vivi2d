# 파일 형식

## 번역 초안 안내

이 페이지는 번역 초안입니다. 가장 최신이고 자세한 절차는 같은 주제의 영어판에 먼저 반영됩니다. 공개 전에는 이 번역도 내용을 확장하고 다시 검토합니다.

테스트하려는 워크플로에 맞춰 입력과 출력 파일을 선택합니다.

## 지원 입력

공개 전에는 지원 상태가 바뀔 수 있으므로 현재 앱 UI를 최종 기준으로 삼으세요. 일반적으로 Vivi2D 흐름은 편집 가능한 레이어나 승인된 마스크가 될 수 있는 아트워크 파일을 중심으로 합니다.

## 좋은 원본 아트워크

- 가져오기 전에 원본 아트워크 사본을 보관합니다.
- 레이어가 있는 원본은 명확한 레이어 이름이 좋습니다.
- 한 장의 그림은 Auto Setup 전에 수동 이미지 분할이 필요할 수 있습니다.
- 공개 버그 보고서나 스크린샷에 비공개 의뢰 작업물을 사용하지 마세요.

## 출력 파일

공유하기 전에 내보낸 파일을 의도한 Viewer나 샘플에서 테스트합니다. 출력 경로가 실험적이라고 표시되면 릴리스 노트에 명시되기 전까지 테스트 산출물로 다룹니다.

## 문제가 있을 때

확장자와 실제 내용이 맞지 않으면 가져오기가 실패할 수 있습니다. 안전한 복구 단계는 [가져오기 오류](../troubleshooting/import-errors.md)를 확인하세요.
