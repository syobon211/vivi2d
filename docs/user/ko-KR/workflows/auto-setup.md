# Auto Setup

## 번역 초안 안내

이 페이지는 번역 초안입니다. 가장 최신이고 자세한 절차는 같은 주제의 영어판에 먼저 반영됩니다. 공개 전에는 이 번역도 내용을 확장하고 다시 검토합니다.

무엇이 저장되고 무엇이 미리보기 전용으로 남는지 확인하면서 시작 설정을 만듭니다.

## 필요한 것

- 분리된 레이어 또는 승인된 수동 분할 마스크.
- 준비 단계에 남아 있지 않은 차단 품질 경고.
- 결과가 맞지 않을 때 돌아갈 수 있는 저장된 프로젝트 사본.

## 단계

1. 레이어 구조가 준비된 뒤 Auto Setup을 엽니다.
2. 원하는 제어 정도에 따라 초보자 또는 고급 감지 수준을 선택합니다.
3. 적용하기 전에 생성된 계획을 검토합니다.
4. 저장되는 작업 영역을 읽습니다. 여기에는 본, 정적 가중치, 물리 그룹, 정리 요약, 수치 설정처럼 프로젝트 데이터가 될 수 있는 항목이 포함됩니다.
5. 버려지는 미리보기 영역을 읽습니다. 임시 미리보기 정보와 진단 세부 정보는 프로젝트에 저장되지 않습니다.
6. 보호 영역, 중복 윤곽, 숨겨진 부분 노출, 완료되지 않은 검사에 대한 경고를 확인합니다.
7. 저장 작업과 경고가 예상과 맞을 때만 적용합니다.

## 결과 확인

프로젝트에는 계속 편집할 수 있는 명확한 시작 설정이 들어 있어야 합니다. 검토 패널은 저장된 것, 버려진 것, 수동 확인이 필요한 것을 설명해야 합니다.

## 문제가 있을 때

잘못된 영역이 움직이거나 얼굴 변화가 크거나 수동 검토가 필요하면 적용하지 말고 취소하세요. 원본 레이어나 마스크를 고친 뒤 Auto Setup을 다시 실행합니다.
