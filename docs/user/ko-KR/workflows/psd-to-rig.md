# 아트워크에서 시작 리그까지

## 번역 초안 안내

이 페이지는 번역 초안입니다. 가장 최신이고 자세한 절차는 같은 주제의 영어판에 먼저 반영됩니다. 공개 전에는 이 번역도 내용을 확장하고 다시 검토합니다.

검토 단계를 건너뛰지 않고 원본 아트워크에서 처음 편집 가능한 설정으로 이동합니다.

## 필요한 것

- 캐릭터 파트를 따로 확인할 수 있는 아트워크.
- 저장된 프로젝트 사본.
- 생성된 설정 변경을 적용하기 전에 경고를 검토할 시간.

## 단계

1. 아트워크를 열고 캔버스, 레이어, 속성 패널이 올바른지 확인합니다.
2. 이미 레이어가 나뉜 경우 레이어 이름, 표시 상태, 순서를 확인합니다.
3. 한 장의 그림이라면 먼저 [수동 이미지 분할](manual-image-split.md)로 승인된 마스크를 만듭니다.
4. 레이어나 마스크가 예상한 캐릭터 파트와 맞을 때 [Auto Setup](auto-setup.md)을 실행합니다.
5. 적용하기 전에 Auto Setup이 저장할 내용을 확인합니다.
6. [Viewer](viewer.md)에서 결과를 미리 보고, 움직임이나 분리가 어색하면 이전 단계로 돌아갑니다.

## 결과 확인

편집 가능한 레이어, 검토한 시작 설정, 적용 전에 확인한 경고가 있는 상태여야 합니다.

## 문제가 있을 때

정리되지 않은 입력에 Auto Setup을 반복해서 적용하지 마세요. 레이어 이름, 마스크, 수동 정리 단계로 돌아가고 깨끗한 프로젝트 사본에서 다시 실행합니다.
