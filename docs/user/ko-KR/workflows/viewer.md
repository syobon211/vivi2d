# Viewer에서 미리보기

## 번역 초안 안내

이 페이지는 번역 초안입니다. 가장 최신이고 자세한 절차는 같은 주제의 영어판에 먼저 반영됩니다. 공개 전에는 이 번역도 내용을 확장하고 다시 검토합니다.

편집 작업 공간 밖에서 모델이 어떻게 보이는지 Viewer로 확인합니다.

## 필요한 것

- Viewer가 불러올 수 있는 저장된 Vivi2D 프로젝트 또는 내보낸 모델 파일.
- 윤곽을 확인하기 쉬운 단순한 배경.
- 표정, 파라미터, 기본 동작을 테스트할 시간.

## 단계

1. 앱이나 관련 워크플로에서 Viewer를 엽니다.
2. 확인할 모델을 불러옵니다.
3. 먼저 중립적인 배경을 사용해 중복 윤곽과 숨겨진 부분 노출을 찾기 쉽게 합니다.
4. 움직임이나 표정을 테스트하기 전에 정지 상태를 확인합니다.
5. 사용할 수 있는 컨트롤을 시도한 뒤 정지 상태로 되돌아가는지 확인합니다.
6. Viewer에 경고가 보이면 내보내기 전 검토 작업으로 다룹니다.

## 결과 확인

모델이 불러와지고, 조작에 반응하며, 눈에 띄는 누락이나 예상치 못한 윤곽 없이 안정적인 정지 상태로 돌아와야 합니다.

## 문제가 있을 때

Viewer가 비어 보이면 편집기로 돌아가 모델이 올바르게 저장되었거나 내보내졌는지 확인합니다. 표시 문제라면 [표시와 GPU](../troubleshooting/display-and-gpu.md)를 읽어 보세요.
