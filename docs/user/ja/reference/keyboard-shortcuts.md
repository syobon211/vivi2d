# キーボードショートカット

## 翻訳ドラフトについて

このページは翻訳ドラフトです。最新かつ最も詳しい手順は、同じトピックの英語版に先に反映されます。公開前に、この翻訳も内容を広げてレビューします。

ショートカットは操作を速くする補助です。操作へ到達する唯一の方法として扱わないでください。

## よく使う領域

- 取り消しとやり直しは、編集ミスから戻るために使います。
- ツール切り替えは、現在の編集モードを変えます。
- レイヤー移動は重なり順を変えるため、慎重に使います。
- 保存と別名保存のショートカットは、定期的なバックアップの代わりにはなりません。

## カスタマイズ

設定メニューからショートカット設定を開きます。OS、ブラウザー、支援技術、入力デバイスと競合する場合は、別のキーに変えるか、画面上のボタンを使ってください。

## 結果を確認する

ショートカット設定ダイアログには、翻訳された操作名と安定したキー割り当てが表示されます。変更後はプロジェクトコピーで試します。

## うまくいかないとき

ショートカットが反応しない場合は、まず画面上の操作をクリックします。その後、別のアプリや OS がキーを取得していないか確認してください。
