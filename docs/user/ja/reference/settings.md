# 設定

## 翻訳ドラフトについて

このページは翻訳ドラフトです。最新かつ最も詳しい手順は、同じトピックの英語版に先に反映されます。公開前に、この翻訳も内容を広げてレビューします。

設定は、プロジェクトデータを変えずにワークスペースを読みやすくするために使います。

## よく使う設定

- 言語は、エディターまたは Viewer の該当画面の UI 言語を変えます。
- テーマは、ワークスペースの見た目を変えます。
- ワークフロー設定は、パネルやレビュー補助の表示に影響することがあります。
- ショートカット設定は、画面上のボタンとは別にキーボード割り当てを管理します。

## 言語について

公開向けの初期 UI は英語です。現在のローカライズ対象は、日本語、簡体字中国語、韓国語です。言語を変えても英語のラベルが残る場合は、ユーザーの操作ミスではなくローカライズの問題として扱います。

## 結果を確認する

設定を変えた後、メニューやダイアログを開いて新しい選択が反映されているか確認します。一部の設定は該当ウィンドウを開き直す必要があります。

## うまくいかないとき

文字化け、文字切れ、言語の混在が見える場合は [言語と文字表示](../troubleshooting/localization.md) を確認してください。
