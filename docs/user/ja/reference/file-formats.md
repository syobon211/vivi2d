# ファイル形式

## 翻訳ドラフトについて

このページは翻訳ドラフトです。最新かつ最も詳しい手順は、同じトピックの英語版に先に反映されます。公開前に、この翻訳も内容を広げてレビューします。

試したいワークフローに合わせて、入力ファイルと出力ファイルを選びます。

## 対応入力

公開前は対応状況が変わる可能性があるため、最終的な確認は現在のアプリ UI を基準にしてください。基本的に Vivi2D のワークフローは、編集可能なレイヤーや承認済みマスクにできる素材を中心に扱います。

## 使いやすい素材

- 読み込む前に元素材のコピーを残します。
- レイヤー素材では、分かりやすいレイヤー名を優先します。
- 1枚絵では、Auto Setup の前に手動画像分割が必要になることがあります。
- 公開の不具合報告やスクリーンショットに、非公開の依頼素材を使わないでください。

## 出力ファイル

出力したファイルは、共有前に想定する Viewer やサンプルで確認します。実験的と表示される出力先は、リリースノートで明記されるまでテスト用として扱ってください。

## うまくいかないとき

拡張子と実際の内容が一致しない場合、読み込みに失敗することがあります。安全な復旧手順は [読み込みエラー](../troubleshooting/import-errors.md) を確認してください。
