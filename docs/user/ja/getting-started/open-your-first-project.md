# 最初のプロジェクトを開く

## 翻訳ドラフトについて

このページは翻訳ドラフトです。最新かつ最も詳しい手順は、同じトピックの英語版に先に反映されます。公開前に、この翻訳も内容を広げてレビューします。

安全なテスト用コピーを開き、変更を始める前にエディターの状態を確認します。

## 用意するもの

- 試したい素材のコピー。
- Vivi2D が現在対応している形式のファイル。
- セットアップツールを使う前に、レイヤー一覧とキャンバスを確認する時間。

## 手順

1. ファイルメニューから、テスト用の素材を開くか読み込みます。
2. 他のツールをクリックする前に、キャンバスの更新が終わるまで待ちます。
3. レイヤー一覧を確認します。一般的な名前よりも、部位名が付いたレイヤーのほうが確認しやすくなります。
4. レイヤーを選択し、プロパティパネルが更新されることを確認します。
5. キャンバスをズーム、パンして、素材が表示され位置が合っていることを確認します。
6. 手動画像分割や Auto Setup を実行する前に、プロジェクトのコピーを保存します。

## 結果を確認する

キャンバスに素材が表示され、レイヤー一覧に読み込まれたパーツが並び、レイヤーを選ぶとプロパティパネルが反応すれば成功です。

## うまくいかないとき

ファイルが開かない場合は、小さな既知の正常ファイルで試し、[読み込みエラー](../troubleshooting/import-errors.md) を確認してください。レイヤーはあるのにキャンバスが空の場合は [表示と GPU](../troubleshooting/display-and-gpu.md) を確認します。
