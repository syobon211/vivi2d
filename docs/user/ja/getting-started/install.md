# Vivi2D をインストールする

## 翻訳ドラフトについて

このページは翻訳ドラフトです。最新かつ最も詳しい手順は、同じトピックの英語版に先に反映されます。公開前に、この翻訳も内容を広げてレビューします。

手元の Vivi2D で利用できる、いちばん信頼できる起動方法を使います。

## 用意するもの

- 現代的なデスクトップアプリを動かせるコンピューター。
- Vivi2D の公式ビルド、またはローカル開発用のリポジトリ。
- 元の素材とテストプロジェクトを分けて保存できる作業フォルダー。

## 手順

1. 公式のパッケージ版がある場合は、プロジェクトの公開元から取得し、アーカイブ名を変えずに保管します。
2. まだパッケージ版がない場合は、非公式ミラーではなく README の開発ビルド手順を使います。
3. 一時ダウンロードフォルダーではなく、通常のユーザーフォルダーから Vivi2D を起動します。
4. テスト用プロジェクトに読み込む前に、元の素材をコピーしておきます。
5. 実作業の前に一度起動し、ウィンドウタイトル、テーマ、言語設定を確認します。

## 結果を確認する

Vivi2D が空のエディター画面で開けば成功です。公開向けの初期表示は英語で、設定を変えない限りダークテーマを使います。

## うまくいかないとき

起動しない場合は、権限の特殊なフォルダーを避け、OS がダウンロードしたアプリをブロックしていないか確認してください。報告するときは、非公開素材やローカルのファイルパスを共有しないでください。
