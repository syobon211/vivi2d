# 初回起動

## 翻訳ドラフトについて

このページは翻訳ドラフトです。最新かつ最も詳しい手順は、同じトピックの英語版に先に反映されます。公開前に、この翻訳も内容を広げてレビューします。

プロジェクトを開く前に、空のワークスペースが何を表しているか確認します。

## 用意するもの

- インストール済み、または開発ビルドから起動した Vivi2D。
- 初期画面を確認するための数分。

## 手順

1. Vivi2D を起動し、エディターウィンドウの読み込みが終わるまで待ちます。
2. アプリのタイトルが Vivi2D Editor になっていることを確認します。
3. 公開向けの初期 UI は英語で、初期テーマはダークです。
4. 日本語、簡体字中国語、韓国語に変えたい場合は設定メニューを開きます。
5. キャンバス、レイヤー一覧、プロパティ、パラメーター、下部のタイムライン領域を確認します。
6. キャンバスが空でも問題ありません。これはプロジェクト未読み込みの正常な状態です。

## 結果を確認する

プロジェクトが読み込まれていない安定したエディター画面が表示されます。素材を開くまでは、ファイルエラーは表示されないはずです。

## うまくいかないとき

文字が切れる、または言語が混ざって見える場合は、一度言語を変更し、必要なら再起動して [言語と文字表示](../troubleshooting/localization.md) を確認してください。
