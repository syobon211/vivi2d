# 手動画像分割

## 翻訳ドラフトについて

このページは翻訳ドラフトです。最新かつ最も詳しい手順は、同じトピックの英語版に先に反映されます。公開前に、この翻訳も内容を広げてレビューします。

1枚絵を Auto Setup の前に分けたいときは、手動マスクで部位を準備します。

## 用意するもの

- 1枚絵、または一部が結合された素材のコピー。
- 髪、顔、目、口、体、小物など、重要な領域の見通し。
- 分割を適用する前に警告を確認する時間。

## 手順

1. 元素材を含むプロジェクトから、手動画像分割のワークスペースを開きます。
2. まず主要な部位のマスクを作り、そのあと細部を整えます。
3. 選択する輪郭に合わせて、ブラシまたは投げ縄ツールを使います。髪や布の輪郭では、滑らかな投げ縄パスが役立ちます。
4. 顔、目、口の領域をきれいに保ちます。動かす髪や小物のマスクに、これらが混ざらないようにします。
5. 空のマスク、重なり、不透明ピクセルの欠落、小さな島などの品質警告を確認します。
6. 後続ツールに使わせたい部位としてマスクが納得できる状態になってから適用します。

## 結果を確認する

承認済み分割によって、重要なピクセルが隠れず、目立つ二重線も少ない、使いやすいレイヤー領域が作られます。

## うまくいかないとき

手動確認が必要という警告が出た場合は、適用する前にマスクを修正してください。プレビュー用の下地補完は自動保存されず、承認したクリーンアップだけがプロジェクトに入ります。
