# Auto Setup

## 翻訳ドラフトについて

このページは翻訳ドラフトです。最新かつ最も詳しい手順は、同じトピックの英語版に先に反映されます。公開前に、この翻訳も内容を広げてレビューします。

何が保存され、何がプレビュー専用のまま残るのかを確認しながら、開始用セットアップを作ります。

## 用意するもの

- 分離済みレイヤー、または承認済みの手動分割マスク。
- 準備段階で残っていない重大な品質警告。
- 結果が合わないときに戻れる保存済みプロジェクトコピー。

## 手順

1. レイヤー構造が整ってから Auto Setup を開きます。
2. 必要な制御量に合わせて、初心者向けまたは詳細な検出レベルを選びます。
3. 適用する前に、生成された計画を確認します。
4. 保存される操作を読みます。ここには、ボーン、静的な重み、物理グループ、クリーンアップ要約、数値設定など、プロジェクトデータになるものが含まれます。
5. 破棄されるプレビュー情報を読みます。一時的なプレビュー情報と診断の詳細はプロジェクトに保存されません。
6. 保護領域、二重線、隠れていた部分の露出、不完全なチェックに関する警告を確認します。
7. 保存される操作と警告が期待どおりのときだけ適用します。

## 結果を確認する

プロジェクトには編集可能な開始用セットアップが入ります。レビューパネルでは、保存されたもの、破棄されたもの、手動確認が必要なものが分かるはずです。

## うまくいかないとき

違う領域が動く、顔の変化が大きい、手動確認が必要と言われる場合は、適用せずキャンセルします。元のレイヤーやマスクを直してから再実行してください。
