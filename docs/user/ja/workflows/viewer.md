# Viewer で確認する

## 翻訳ドラフトについて

このページは翻訳ドラフトです。最新かつ最も詳しい手順は、同じトピックの英語版に先に反映されます。公開前に、この翻訳も内容を広げてレビューします。

編集ワークスペースの外でモデルがどう見えるかを Viewer で確認します。

## 用意するもの

- Viewer が読み込める保存済み Vivi2D プロジェクト、または出力済みモデル。
- 輪郭を確認しやすいシンプルな背景。
- 表情、パラメーター、基本的な動きを試す時間。

## 手順

1. アプリ内、または該当ワークフローから Viewer を開きます。
2. 確認したいモデルを読み込みます。
3. まずは中立的な背景を使い、二重線や隠れていた部分の露出を見つけやすくします。
4. 動きや表情を試す前に、静止状態を確認します。
5. 利用できる操作を試し、静止状態へ戻してきれいに戻ることを確認します。
6. Viewer に警告が出る場合は、出力前の確認作業として扱います。

## 結果を確認する

モデルが読み込まれ、操作に反応し、欠けや予期しない輪郭なしに安定した静止状態へ戻れば成功です。

## うまくいかないとき

Viewer が空に見える場合は、エディターに戻ってモデルが正しく保存または出力されているか確認します。表示に関する問題なら [表示と GPU](../troubleshooting/display-and-gpu.md) を確認してください。
