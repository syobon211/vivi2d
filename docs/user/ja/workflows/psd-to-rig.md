# 素材から開始用リグへ

## 翻訳ドラフトについて

このページは翻訳ドラフトです。最新かつ最も詳しい手順は、同じトピックの英語版に先に反映されます。公開前に、この翻訳も内容を広げてレビューします。

確認手順を飛ばさずに、元素材から編集可能な開始用セットアップへ進みます。

## 用意するもの

- キャラクターの部位を確認しやすい素材。
- 保存済みのプロジェクトコピー。
- 生成された変更を適用する前に警告を確認する時間。

## 手順

1. 素材を開き、キャンバス、レイヤー、プロパティパネルが正しいことを確認します。
2. 既にレイヤー分けされている場合は、レイヤー名、表示状態、順序を確認します。
3. 1枚絵の場合は、先に [手動画像分割](manual-image-split.md) で承認済みマスクを作ります。
4. レイヤーまたはマスクが想定した部位と合ってから [Auto Setup](auto-setup.md) を実行します。
5. Auto Setup が保存する内容を確認してから適用します。
6. [Viewer](viewer.md) で結果を確認し、動きや分離が不自然なら前の手順に戻ります。

## 結果を確認する

編集可能なレイヤー、確認済みの開始用セットアップ、適用前に見落としていない警告がある状態になっていれば成功です。

## うまくいかないとき

入力が崩れている状態で Auto Setup を何度も重ねないでください。レイヤー名、マスク、手動修正に戻り、きれいなプロジェクトコピーから再実行します。
