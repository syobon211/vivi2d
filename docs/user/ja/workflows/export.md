# 出力

## 翻訳ドラフトについて

このページは翻訳ドラフトです。最新かつ最も詳しい手順は、同じトピックの英語版に先に反映されます。公開前に、この翻訳も内容を広げてレビューします。

エディターと Viewer で確認してから出力します。

## 用意するもの

- 重大なレビュー警告が残っていない保存済みプロジェクト。
- 元素材を上書きしない出力先フォルダー。
- プレビュー、共有、連携テストなど、目的となるワークフロー。

## 手順

1. 出力操作を開く前にプロジェクトを保存します。
2. 試したいワークフローに合う出力先を選びます。
3. アプリに表示される実験的機能や互換性に関する注意を読みます。
4. 結果を確認しやすいように、新しいフォルダーへ出力します。
5. 共有する前に、想定する Viewer やサンプルで出力結果を開きます。

## 結果を確認する

出力ファイルがそろっていて、名前が分かりやすく、非公開の作業フォルダーにあるファイルなしで対象ワークフローに読み込めれば成功です。

## うまくいかないとき

テクスチャ、レイヤー、動きが欠けている場合は、元プロジェクトを確認してから再出力します。比較用に正常な出力を1つ残しておくと便利です。
