# 言語と文字表示

## 翻訳ドラフトについて

このページは翻訳ドラフトです。最新かつ最も詳しい手順は、同じトピックの英語版に先に反映されます。公開前に、この翻訳も内容を広げてレビューします。

ラベル、メニュー、ダイアログが選択した言語と合わないときに使います。

## 用意するもの

- 現在のアプリ言語設定。
- 文字がおかしく見える画面またはダイアログ。
- 未翻訳、文字切れ、読めない文字のどれに近いかの短い説明。

## 手順

1. 設定を開き、選択中の言語を確認します。
2. 一度別の言語へ切り替え、もう一度使いたい言語へ戻します。
3. 問題が出ていたダイアログやパネルを開き直します。
4. 直らない場合は、見えているラベルとページまたはパネル名を控えます。
5. レイアウト報告にスクリーンショットが必要な場合は、非公開素材を隠した合成画像や切り抜きを使います。

## 結果を確認する

メニュー、ダイアログ、ワークフローパネル、よく使う操作ラベルに選択中の言語が反映されます。

## うまくいかないとき

開発中は、一部の新しい UI や draft ページに英語が残ることがあります。報告するときは、見えているラベルと言語を伝え、ローカルパスや非公開プロジェクト情報は含めないでください。
