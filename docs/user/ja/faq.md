# FAQ

## 翻訳ドラフトについて

このページは翻訳ドラフトです。最新かつ最も詳しい手順は、同じトピックの英語版に先に反映されます。公開前に、この翻訳も内容を広げてレビューします。

詳しいワークフローを読む前に、短く確認したいときに使います。

## Vivi2D は公開制作で使える状態ですか？

プロジェクトは公開準備中です。リリースノートで明記されるまでは、現在のビルドとドキュメントを draft として扱ってください。

## どの言語を使えばよいですか？

公開向けの初期 UI は英語です。日本語、簡体字中国語、韓国語もユーザー向け UI とドキュメントで利用できますが、最終的なネイティブ話者レビューは公開後の意見も含めて進める場合があります。

## Manual Image Split と Auto Setup のどちらから始めますか？

1枚絵なら手動画像分割から始めます。既に分かりやすいレイヤーがある素材なら、レイヤーを確認してから Auto Setup を試します。

## Auto Setup は何を保存しますか？

Auto Setup は、確認済みのプロジェクト操作として、ボーン、静的な重み、物理グループ、クリーンアップ要約、数値設定などを保存します。一時的なプレビュー計算や診断の詳細は破棄されます。

## 外部ツールから Viewer を操作できますか？

ローカルの Viewer API の流れで操作できます。ペアリングを行い、スコープを確認し、テスト後はセッションを閉じるか取り消してください。
