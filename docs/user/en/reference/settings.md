# Settings

Use settings to make the workspace easier to read without changing project data.

## Common Settings

- Language changes the UI language for the editor or viewer surface that owns the setting.
- Theme changes the visual appearance of the workspace.
- Workflow preferences can affect how panels and review helpers appear.
- Shortcut settings control keyboard bindings separately from visual buttons.

## Language Notes

The default public UI is English. Japanese, Simplified Chinese, and Korean are available in the current localization set. If a label remains in English after switching language, it should be treated as a localization issue, not as user error.

## Check Your Result

After changing settings, open a menu or dialog and confirm the new choice is visible. Some settings may require reopening the relevant window.

## If Something Looks Wrong

If text looks garbled, clipped, or mixed between languages, see [Language And Text Display](../troubleshooting/localization.md).
