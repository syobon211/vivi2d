# Keyboard Shortcuts

Use shortcuts as accelerators, not as the only way to reach an action.

## Common Areas

- Undo and redo help recover from editing mistakes.
- Tool switching changes the active edit mode.
- Layer movement changes stacking order, so use it carefully.
- Save and save-as shortcuts should not replace regular project backups.

## Customization

Open shortcut settings from the settings menu. If a shortcut conflicts with your operating system, browser, assistive technology, or input device, choose a different binding or use the visible button instead.

## Check Your Result

The shortcut settings dialog should show localized action names and stable key bindings. After changing a shortcut, test it on a copy of the project.

## If Something Looks Wrong

If a shortcut does nothing, click the visible control first. Then check whether another app or the operating system is capturing the key.
