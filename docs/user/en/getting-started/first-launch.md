# First Launch

The first launch is a safe place to learn the editor layout before a project is loaded.

## What You Should See

The empty editor is not an error. It means Vivi2D is ready for you to open or import artwork.

Look for these areas:

- **Top menu and toolbar:** file actions, tools, view controls, settings, and integrations.
- **Canvas:** the central workspace where artwork and setup previews appear.
- **Layer list:** the left-side list of imported or created parts.
- **Properties panel:** the right-side details for the selected layer, model, or tool.
- **Parameters and timeline area:** the lower area used for parameter values, clips, and preview controls.
- **Status and warnings:** messages that explain why an action is disabled or needs review.

## Default Settings

- The default public UI language is English.
- The default theme is dark.
- Japanese, Simplified Chinese, and Korean are available from Settings.
- Language changes should affect menus, dialogs, workflow panels, and common action labels.

If you see mixed language in common UI, note the visible label and the locale, then check [Language And Text Display](../troubleshooting/localization.md).

## Safe First Actions

1. Open the File menu, but do not open real artwork yet.
2. Open Settings and confirm the current language and theme.
3. Open the shortcut settings dialog if you want to see available actions.
4. Select different empty panels and notice how the Properties panel changes.
5. Close any dialogs before opening artwork.

## What Not To Worry About

- A blank canvas is normal before loading a project.
- A message such as “No project loaded” is normal.
- Empty parameter or timeline panels are normal.
- Some optional integrations may be hidden if the build does not include them.

## Check Your Result

You are ready to continue when:

- The editor window is stable.
- You know where the canvas, layers, properties, and settings are.
- You can open and close a dialog without losing the main workspace.
- You understand that no project data exists yet.

## If Something Looks Wrong

- If the window is blank, wait a few seconds, then restart Vivi2D.
- If text is unreadable, change language once, reopen the dialog, and restart if needed.
- If the app opens on the wrong monitor or scale, check your OS display scaling and [Display And GPU](../troubleshooting/display-and-gpu.md).
- If a dialog is too small, try a larger window before reporting layout clipping.
