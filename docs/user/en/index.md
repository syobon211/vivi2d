# Vivi2D User Guide

This guide helps artists, riggers, streamers, and SDK users understand the current Vivi2D workflow without needing repository knowledge.

## Start Here

1. Read [Install Vivi2D](getting-started/install.md) to understand the current release status.
2. Continue with [First Launch](getting-started/first-launch.md) to learn the default language, dark theme, and empty workspace state.
3. Open artwork with [Open Your First Project](getting-started/open-your-first-project.md).
4. Follow [From Artwork To Starting Rig](workflows/psd-to-rig.md) for the complete creation path.

## Main Workflows

- [Manual Image Split](workflows/manual-image-split.md) prepares flat artwork for setup.
- [Auto Setup](workflows/auto-setup.md) creates a reviewed starting point from prepared layers.
- [Preview In Viewer](workflows/viewer.md) checks the model before export or integration.
- [Viewer API Basics](workflows/viewer-api.md) explains local pairing for external control.

## Optional Integrations

- [ComfyUI Setup](integrations/comfyui.md) explains how to connect a local ComfyUI workspace while keeping generated results behind Vivi2D review steps.
